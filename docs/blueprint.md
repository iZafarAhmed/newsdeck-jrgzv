# **App Name**: NewsDeck

## Core Features:

- Categorized News Display: Display news categories with links to external news sources.
- Category Filtering: Enable users to filter news sources by category.
- Source Search: Allow users to search for specific news sources using a search bar.
- Responsive Layout: Provide a mobile-responsive design for access on various devices.
- Enhanced UX: Implement smooth scrolling and loading states for a better user experience.
- Smart Article Suggestion: AI tool that identifies a single news article of interest based on the previous search and filtering that has been performed.

## Style Guidelines:

- Primary color: Slate Blue (#6A5ACD), a sophisticated, muted blue conveying trust and reliability.
- Background color: Light Gray (#F0F0F0), a neutral backdrop for readability and focus on content.
- Accent color: Gold (#FFD700), used sparingly to highlight interactive elements and important information.
- Body and headline font: 'Inter', a sans-serif font providing a clean and modern reading experience.
- Use a set of modern, flat design icons from Font Awesome to represent different news categories.
- Grid-based layout with clear separation of news categories to enhance scannability.
- Subtle fade-in animations on category cards and news source links to improve user engagement.
